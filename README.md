# cory_boyu
《FindFun》：一款基于Rxjava+RetroFit的简单APP。

/********************/
主要功能模块模块:
(1)新闻界面
(2)段子界面
(3)趣图界面
(4)历史界面
(5)关于界面

/********************/
用到的一些开源库：
(1)数据来源： 聚合数据
(2)图片加载框架： Glide
(3)网络请求框架： 鸿洋okhttputils
(4)view注解框架： butterknife
(5)基类适配器： BaseRecyclerViewAdapterHelper
(6)指示器： MagicIndicator
(7)底部栏： BoomMenu
